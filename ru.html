jQuery UI Sortable Animation

Простой способ добавить анимацию сортировки на ваш сайт, использующий jQuery UI, в 1 строку кода.

Описание

У вас есть сайт, использующий библиотеку jQuery UI, но на данный момент сортировка .sortable() не поддерживает анимацию элементов "из коробки". С помощью данного небольшого плагина это легко исправляется, надо лишь добавить 2 специальных параметра в вызов .sortable().

Настройка

Доступные параметры:

  • animation - длительность анимации, 0 без анимации

Пример кода

$('#sortable').sortable({
  axis: 'y',
  containment: 'parent',
  animation: 200,
});
